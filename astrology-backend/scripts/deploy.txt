Monorepo root: https://github.com/abdulkarimtaji33/astrology.git

Server (live): git clone lives at /var/www/astrology
  /var/www/astrology-backend  -> symlink -> /var/www/astrology/astrology-backend
  /var/www/astrology-frontend -> symlink -> /var/www/astrology/astrology-frontend

Deploy:
  cd /var/www/astrology
  git pull
  git lfs pull
  cd astrology-backend && npm ci && npm run build && npm prune --omit=dev
  cd ../astrology-frontend && npm ci && npm run build
  pm2 restart astrology-api astrology-web --update-env

DB import (when astrology.sql changes):
  bash astrology-backend/scripts/import-db.sh

Git LFS: one-time on server: git lfs install
